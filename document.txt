Read this file before run the program.
structure:
--root
    |--model
    |--raw
    |--result
    |--split
    |--main.py
    |--document.txt
All of this folder and main.py must be placed together like above.
You can clone the whole program from github or you can set up the env manually.
-------------------------------------------------------------------------------------------
folder 'raw' will store all the uploaded images
folder 'split' will store all cropped images. This folder will have several folder under it.
folder 'model' stores weights and class file. Do not touch it
folder 'result' will store outputs, including visual images and text files.
file 'main.py' is the program.
